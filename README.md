Descripcion del taller:
Este repositorio contiene la solución de un taller de programación en Java
desarrollado en NetBeans y ejecutado desde consola.

El taller incluye 15 ejercicios enfocados en el manejo de estructuras de datos
básicas en Java:

- Arreglos (arrays)
- ArrayList
- Matrices
- Lectura de archivos
- Programación orientada a objetos

Cada ejercicio se implementa en una clase independiente y se ejecuta desde
clases principales (View) que permiten probar los ejercicios desde la consola.

También se aplican principios básicos de programación como:
- KISS
- DRY
- Organización del código mediante métodos para generación de datos,
  procesamiento e impresión de resultados.
  -----------------------------------------------------------------------------------------
INSTTRUCCIONES PARA EJECUTAR EL PROGRAMA
1. Clonar el repositorio o descargar el proyecto.

2. Abrir el proyecto en NetBeans.

3. Ubicar las clases principales según el tipo de ejercicio:

   - ViewArreglo → ejecuta los ejercicios de arreglos.
   - ViewArrayList → ejecuta los ejercicios de ArrayList.
   - ViewMatrix → ejecuta los ejercicios de matrices.
   - ArraylistObjeto → ejecuta el ejercicio que lee el archivo
     PartidoLiga.txt.

4. Ejecutar la clase principal deseada con la opción "Run File".

5. El programa se ejecutará en consola mostrando los resultados de
   cada ejercicio.

Nota:
El archivo `PartidoLiga.txt` debe estar en la misma carpeta del
proyecto para que el programa pueda leer los datos correctamente.
